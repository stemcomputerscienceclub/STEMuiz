FAVICON INSTRUCTIONS

To complete the STEMuiz branding, please generate and add the following favicon files to the public directory:

1. favicon.ico - Main favicon file
2. apple-touch-icon.png - 180×180px icon for iOS devices
3. favicon-32x32.png - 32×32px favicon
4. favicon-16x16.png - 16×16px favicon
5. android-chrome-192x192.png - 192×192px icon for Android
6. android-chrome-512x512.png - 512×512px icon for Android

You can generate these files using online tools such as:
- https://realfavicongenerator.net/
- https://favicon.io/

Design recommendation:
- Use a simple "S" or atom/science-related icon
- Use the indigo color palette (#4f46e5) for consistency with the app's theme
- Ensure the design is clear even at small sizes
